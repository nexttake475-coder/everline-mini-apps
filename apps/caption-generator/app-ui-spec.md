# App UI Spec — Caption Generator

## Product
Caption Generator

## Company
Everline Media Systems

---

# UI Objective

Design a simple, premium-feeling interface that helps users generate stronger captions quickly.

The interface should feel:
- focused
- fast
- lightweight
- intuitive
- mobile-friendly

The product should not feel like:
- a cluttered dashboard
- a generic chatbot
- a complicated software suite

---

# UX Principle

The workflow should be:

idea → inputs → generate → usable output

with as little friction as possible.

The user should understand the product in under 15 seconds.

---

# Mobile Design

The mobile experience is critical.

## Mobile Rules
- stacked layout
- large tap targets
- readable typography
- no horizontal scrolling
- full-width buttons
- comfortable spacing

The app should feel smooth on iPhone screens.

---

# Desktop Design

## Desktop Rules
- centered layout
- generous spacing
- strong typography hierarchy
- minimal visual clutter

Avoid unnecessary complexity.

---

# Visual Direction

## Style
Sovereign Minimalism

## Design Characteristics
- dark background
- clean typography
- subtle borders
- matte gold accents
- strong spacing
- minimal distractions

---

# Loading State

## Display
"Generating caption..."

Optional:
- spinner
- subtle animation

Do not overcomplicate the loading experience.

---

# Empty State

Before generation, show:

"Enter your topic, platform, and goal to generate a caption."

---

# Error State

If required fields are missing:
- show inline validation

Example:
"Please enter a topic."

If generation fails:
- show:
"Something went wrong. Please try again."
