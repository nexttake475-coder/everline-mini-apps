# Pricing Strategy

Caption Generator should be positioned as:
- a focused utility
- a workflow accelerator
- a lightweight software product

Not as a cheap prompt pack.

The pricing should reflect:
- usability
- workflow value
- repeat usage
- future ecosystem expansion
